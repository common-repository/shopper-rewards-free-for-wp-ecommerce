Import Purchases
This is a manual import option used to create points for existing purchases
Normally you would only have to do an import one time. Use it after you install the plugin to give your past customers points.
If you do an import more than once it should not have have any negative affect. During the import process purchase IDs are checked. Purchase IDs that already have points awarded do not receive points a second time.
If you have turned off the plugin for a period of time and want to make sure points are awarded for purchases during that interval you can click import again.
